Array indexing

Array indexing

Indexing out a time series is done with common bracketing semantics.

Integers

ExampleDescriptionIndexing value
[1]First row of data onlysingle integer
[1:3]First through third row onlyinteger range
[1:2:10]Odd row between first to tenth rowinteger range with step
[[1:3; 8]]First through third row and eight rowinteger range & single integer

Examples in REPL:

julia> using MarketData
ERROR: ArgumentError: Package MarketData not found in current path:
- Run `import Pkg; Pkg.add("MarketData")` to install the MarketData package.

julia> ohlc[1]
ERROR: UndefVarError: ohlc not defined

julia> ohlc[1:3]
ERROR: UndefVarError: ohlc not defined

julia> ohlc[1:2:10]
ERROR: UndefVarError: ohlc not defined

julia> ohlc[[1:3;8]]
ERROR: UndefVarError: ohlc not defined

Strings

ExampleDescriptionIndexing value
["Open"]The column named "Open"single string
["Open", "Close"]The columns named "Open" and "Close"multiple strings

Examples in REPL:

julia> using MarketData
ERROR: ArgumentError: Package MarketData not found in current path:
- Run `import Pkg; Pkg.add("MarketData")` to install the MarketData package.

julia> ohlc["Open"]
ERROR: UndefVarError: ohlc not defined

julia> ohlc["Open", "Close"]
ERROR: UndefVarError: ohlc not defined

Date and DateTime

ExampleDescriptionIndexing value
[Date(2000, 1, 3)]The row containing Jan 3, 2000 timestampsingle Date
[[Date(2000, 1, 3), Date(2000, 2, 4)]]The rows containing Jan 3 & Jan 4, 2000multiple Dates
[Date(2000, 1, 3):Date(2000, 2, 4)]The rows between Jan 1, 2000 & Feb 1, 2000range of Date

Examples in REPL:

julia> using MarketData
ERROR: ArgumentError: Package MarketData not found in current path:
- Run `import Pkg; Pkg.add("MarketData")` to install the MarketData package.

julia> ohlc[Date(2000, 1, 3)]
ERROR: UndefVarError: Date not defined

julia> ohlc[Date(2000, 1, 3):Date(2000, 2, 4)]
ERROR: UndefVarError: Date not defined
